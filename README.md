# Book API

A simple RESTful API built with Node.js and Express to manage a collection of books.

## Features

- Get all books (`GET /books`)
- Add a new book (`POST /books`)
- Update a book by ID (`PUT /books/:id`)
- Delete a book by ID (`DELETE /books/:id`)
